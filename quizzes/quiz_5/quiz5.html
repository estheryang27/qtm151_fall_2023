(d) Flag subset of data and plot

  • Create a boolean colum (True/False) "bool_cond" which checks whether
    "growth_euro" had a value greater than 2 or less than -2
  • Generate a line plot with the "growth_euro" series, label the axes and the title
  • Use the command plt.fill_between() as in Lecture 21 to create a shaded region
    for time periods where "bool_cond" is true.
In[]:
# Write your own code
currencies['bool_cond'] = (currencies['growth_euro']> 2) | (currencies['growth_euro'] < -2)

# Generate a line plot with the 'growth_euro' series
plt.plot("date_parsed", 'growth_euro',
          data = currencies)
plt.xlabel('Date')
plt.ylabel('Growth Euro (%)')
plt.title('Euro to USD Growth Rate')

# Create a shaded region for periods where 'bool_cond' is True
plt.fill_between(x = currencies["date_parsed"], 
                 y1=currencies['growth_euro'].min(), 
                 y2=currencies['growth_euro'].max(), 
                 where=currencies['bool_cond'], 
                 color='red', 
                 alpha=0.2)

plt.show()
